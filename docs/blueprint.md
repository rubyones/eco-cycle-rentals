# **App Name**: eBike Manager

## Core Features:

- Dashboard Overview: Real-time summary of system operations including registered renters, e-bike units, active rentals, and revenue summary.
- E-Bike Management: Register new e-bike units, modify existing records, and remotely lock/unlock e-bikes.
- Station Management: Add or update station information including location and parking bays, with an integrated map interface.
- Renter Management: View renter profiles, current rental status, and rental history. Administrators can suspend or deactivate accounts.
- Rental Monitoring: Monitor active and completed rentals, rental duration, and rental fees. Ability to force rental termination.
- Predictive Maintenance: Tool to generate predictive maintenance schedules based on historical issue data and real-time e-bike status using machine learning.
- Notifications: Configure automated alerts for rental time reminders, payment due notices, and send announcements to renters.

## Style Guidelines:

- Primary color: Teal (#008080) for a professional and trustworthy feel.
- Background color: Light teal (#E0F8F8) for a clean and modern look.
- Accent color: Green (#006400) for CTAs and success indicators.
- Body font: 'Inter' sans-serif for a modern, neutral look.
- Headline font: 'Space Grotesk' sans-serif for a computerized, techy feel.
- Use simple, line-based icons for a consistent and modern look.
- Dashboard should have a clear, tabular format. Ensure clear section delineation with proper spacing and dividers
